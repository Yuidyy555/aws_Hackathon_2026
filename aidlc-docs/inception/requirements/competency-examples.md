## 1. コンピテンシーの体系（Competency Framework）

以下のフレームでコンピテンシーを定義しました。

### 複合スキル × 基礎スキル・マインド（一覧）

複合スキルとその定義に対して、構成となる基礎スキル・マインドと各定義を対応させています。

#### 経営視点（複合）

- **複合スキルの定義**  
高いビジネスリテラシーを持ち、長期視点で全体最適を指向したビジョン創造・問題発見ができる。


| 基礎スキル・マインド（『経営視点』に資する要素） | 定義                              |
| ------------------------ | ------------------------------- |
| 論理的思考力                   | 分析的、論理的に考え抜き、表現できる              |
| ビジネスセンス                  | 収益性を考え、正しい判断や予測を行える             |
| 創造的思考力                   | 前例にとらわれず、仕組みや組織体系を変革するアイディアを出せる |


#### 業務遂行（複合）

- **複合スキルの定義**  
達成すべきゴールを適切に捉え、効率よく円滑にプロジェクト・業務のPDCAを回すことができる。


| 基礎スキル・マインド（『業務遂行』に資する要素） | 定義                        |
| ------------------------ | ------------------------- |
| GRIT（やり抜く力）              | どんな困難があっても目標到達に邁進する       |
| スキル                      | その役割に必要な各種のスキルを具備し、活用している |


#### コラボレーション（複合）

- **複合スキルの定義**  
全ての関係者と情報共有・利害調整を円滑に行い、心理的に安全な環境・風土を整え、相乗効果を生み出しながら物事を推進できる。


| 基礎スキル・マインド（『コラボレーション』に資する要素） | 定義                      |
| ---------------------------- | ----------------------- |
| 建設的議論                        | 自説にこだわりすぎず、よりよい議論を重ねられる |
| 共感力                          | 様々な場面で相手の立場に立って思考、配慮できる |


#### 成長支援（複合）

- **複合スキルの定義**  
個々のメンバーの状況を把握し、成長機会を作り、適切なフィードフォワード・フィードバックができる。


| 基礎スキル・マインド（『成長支援』に資する要素） | 定義                            |
| ------------------------ | ----------------------------- |
| コーチング                    | まず全力で傾聴し、その上で気づきにつながる助言をできる   |
| 育成力                      | 機会提供や適切なFBによって、他者の成長を大きく促進できる |


### ■コンピテンシー定義のポイント（スライド記載）

- 当社らしい組織運営を行うために特に求めたいスキル・マインドを言語化  
  - └ MECE（漏れなく、ダブりなく）に設計しているわけではない
- 「複合スキル」の結果がそのままバンドになるため、バンドごとに求められるレベルを定義する
- 「基礎スキル・マインド」は、当該バンドの期待役割を果たすにあたって強み/弱みとなっているかを他者評価してもらう

## 2. 「マインド」「基礎スキル」の評価基準（スライド表の転記）

当社らしくあるために求めたいスキル・マインドを整理しています。4段階の評価は本人の強み/弱みを振り返るために実施します。

### 評価レベル（全項目共通の考え方）

**レベル（4 > 3 > 2 > 1）**


| レベル | 意味       |
| --- | -------- |
| 4   | 大きな強みである |
| 3   | 基準に達している |
| 2   | 改善の余地あり  |
| 1   | 弱みとなっている |


※ 当該バンドの期待役割を果たすに当たって本人の強み/弱みとなっているかを他者評価してもらう。

### 項目と定義


| 区分               | 項目              | 定義                                   |
| ---------------- | --------------- | ------------------------------------ |
| 『経営視点』に資する要素     | 論理的思考力          | 分析的、論理的に考え抜き、表現できる                   |
| 『経営視点』に資する要素     | ビジネスセンス         | 収益性を考え、正しい判断や予測を行える                  |
| 『経営視点』に資する要素     | 創造的思考力          | 前例にとらわれず、仕組みや組織体系を変革するアイディアを出せる      |
| 『業務遂行』に資する要素     | GRIT（やり抜く力）     | どんな困難があっても目標到達に邁進する                  |
| 『業務遂行』に資する要素     | スキル             | その役割に必要な各種のスキルを具備し、活用している            |
